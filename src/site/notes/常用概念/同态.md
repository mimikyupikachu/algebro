# 代数的同态 
+ 代数的同态满足 
	+ 保持加法: $f(a)+f(b)=f(a+b)$
	+ 保持数乘: $f(ka)=kf(a)$
	+ 保持乘法: $f(a)f(b)=f(ab)$
	+ 保持乘法单位元: $f(e_A)=e_B$

	
+  $\mathrm{End\ }A\cong A$

# 模的同态
+ 模的同态满足
	+ 保持加法: $f(a)+f(b)=f(a+b)$
	+ 保持数乘: $f(ka)=kf(a)$
	+ 保持模作用: $f(am)=af(m)$

 + $\mathrm{End }_A\ M$



# 特别地:

+  代数的自同态和正则模的自同态不一样.
